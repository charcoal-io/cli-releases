# charcoal-cli Releases

Public release repository for the `charcoal-cli`.

Each release includes pre-built Python packages (`.tar.gz` source distribution and `.whl` wheel) along with a changelog describing what changed.

## Installation

```bash
pip install https://github.com/charcoal-io/cli-releases/releases/download/<VERSION>/
charcoal_cli-<VERSION>-py3-none-any.whl
```

Or download the archive from the [releases page](https://github.com/charcoal-io/cli-releases/releases) and install manually:

```bash
pip install ./charcoal_cli-<VERSION>-py3-none-any.whl
```
